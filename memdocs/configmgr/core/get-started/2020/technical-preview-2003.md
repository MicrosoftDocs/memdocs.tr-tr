# <a name="features-in-configuration-manager-technical-preview-version-2003"></a>Configuration Manager Technical Preview Sürüm 2003 ' deki Özellikler

*Uygulama hedefi: Configuration Manager (Technical Preview dalı)*

Bu makalede, Sürüm 2003 Configuration Manager için Technical Preview 'da kullanılabilen özellikler tanıtılmaktadır. Teknik Önizleme sitenize yeni özellikler eklemek ve güncelleştirmek için bu sürümü yükler.

Bu güncelleştirmeyi yüklemeden önce [Teknik Önizleme](../technical-preview.md) makalesini gözden geçirin. Bu makalede, Technical Preview kullanımı, sürümler arasında nasıl güncelleştirme yapılacağı ve geri bildirimde bulunmak için genel gereksinimler ve sınırlamalar tamamladığınızda tercihinize.
