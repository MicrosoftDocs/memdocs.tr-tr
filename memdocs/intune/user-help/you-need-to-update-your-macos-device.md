# <a name="your-macos-device-doesnt-have-the-required-minimum-operating-system-version"></a>macOS cihazınız, gerekli en düşük işletim sistemi sürümüne sahip değil

[!INCLUDE [wit_nextref](includes/end-user-os-update-guidance.md)]

**Uygulama Mağazası** > **güncelleştirmeleri**' ne tıklayarak MacOS cihazınızı güncelleştirin. Mevcut en yeni macOS güncelleştirmesini seçin, ardından **Güncelleştir** düğmesine tıklayın.

Bu bilgiler yardımcı olmadı mı? Şirketinizin destek bölümüne başvurun. Kişi bilgileri için [Şirket Portalı Web sitesine](https://go.microsoft.com/fwlink/?linkid=2010980) bakın.
